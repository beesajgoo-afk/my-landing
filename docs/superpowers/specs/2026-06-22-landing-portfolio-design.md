## 1. Цель и аудитория

**Цель сайта:** Портфолио-лендинг, который работает как одностраничный sales funnel. Посетитель приходит с проблемой — страница отвечает на три вопроса: "Этот человек решит МОЮ проблему? Могу ли я ей доверять? Что мне делать дальше?"

**Автор:** Елена Бисерова — цифровой мастер, создаёт AI-системы, агентов, автоматизации, сайты и тексты для бизнеса.

**Целевая аудитория:** Малый бизнес в России — ИП и небольшие компании, которым нужна автоматизация рутины, современный сайт или AI-инструменты. Нетехническая аудитория, которую не нужно пугать жаргоном.

**Ожидаемый результат:** 3–7% посетителей → обращение (benchmark индустрии: 0.5–1.5%).

## 2. Стек и деплой

| Параметр | Решение |
|----------|---------|
| Стек | HTML5 + CSS3 + Vanilla JS |
| Хостинг | Vercel (бесплатный план) или GitHub Pages |
| CDN-библиотеки | typed.js (7KB), VanillaTilt (3KB), tsParticles (CDN) |
| Шрифты | Google Fonts: Space Grotesk Variable, Inter Variable, JetBrains Mono |
| Аналитика | Plausible или Yandex Metrica (добавить позже) |

**Структура файлов:**
```
my-landing/
├── index.html
├── css/
│   ├── variables.css      (токены: цвета, шрифты, отступы)
│   ├── base.css           (reset, базовая типографика)
│   ├── components.css     (карточки, кнопки, формы, bento)
│   └── animations.css     (scroll-driven, hover, entrance)
├── js/
│   ├── main.js            (курсор, magnetic buttons, spotlight)
│   └── interactions.js    (typed.js init, tilt init, FAQ accordion)
└── assets/
    ├── icons/             (SVG иконки услуг)
    └── images/            (фото, скриншоты кейсов — заглушки)
```

## 4. Структура страницы

### Блок 1 — HERO

**Задача:** Остановить скролл, заявить ценность за 3–5 секунд.

**Контент:**
- **H1:** `"Строю AI-системы и автоматизации, которые убирают рутину и приносят деньги малому бизнесу"`
- **Подзаголовок с typed.js:** `"Елена Бисерова — "` + ротация: `"AI-специалист"` / `"Автоматизатор"` / `"Разработчик"` / `"Копирайтер"`
- **Микросигнал доверия:** `"Работаю с малым бизнесом в России"` (заглушка — заменить на цифры при появлении кейсов)
- **Первичный CTA:** кнопка `"Написать в Telegram"` (фиолетовый, neon glow, magnetic)
- **Вторичная ссылка:** `"Посмотреть работы ↓"` (циан, anchor scroll)

**Визуал:**
- Фон: tsParticles — анимированная сетка точек, реагирует на движение мыши
- Spotlight: `radial-gradient` следует за курсором, открывает сетку фиолетовым светом
- Заглушка фото: абстрактный SVG-аватар до появления реального фото

**Анимации при загрузке:**
1. H1 появляется с анимацией веса: `font-variation-settings: 'wght' 100` → `'wght' 900` за 0.8s
2. Подзаголовок fade-in через 0.5s
3. typed.js запускается через 0.8s
4. CTA-кнопка появляется через 1.0s с `translateY(20px)` → `0`

### Блок 2 — БОЛЬ / ЗЕРКАЛО

**Задача:** Построить эмпатию — клиент чувствует "меня поняли".

**Контент:**
- **Заголовок:** `"Звучит знакомо?"`
- **4 пункта боли:**
  1. `"Сайт устарел, стыдно показывать — а новый всё никак не сделать"`
  2. `"Тратишь часы на рутину: таблицы, пересылки, ручной ввод данных"`
  3. `"Нанимал копирайтеров — они не понимают твой бизнес и пишут мимо"`
  4. `"Слышал про AI, но не знаешь, как реально встроить его в работу"`
- **Переход:** `"Именно с этим приходят ко мне."`

**Визуал:** Тёмный фон с лёгким фиолетовым градиентом снизу — визуальный разрыв от hero, усиливает эмоциональный контраст.

**Анимации:** Каждый пункт боли въезжает слева последовательно при скролле (scroll-driven CSS).

### Блок 4 — ДОКАЗАТЕЛЬСТВА

**Задача:** Убить скептицизм фактами. (Пока заглушки — структура готова к заполнению.)

**Вкладки по типу:** `[Сайты]` `[Автоматизация]` `[AI и агенты]`

**2 карточки кейсов (структура):**
```
Тип клиента:  [Индустрия — заглушка]
Проблема:     [Что мешало работать]
Что сделано:  [Конкретные действия]
Результат:    [Цифра] — например "Сократил ручной труд с 8 часов до 20 минут"
Визуал:       Скриншот / мокап / схема workflow (заглушка)
```

**3 карточки отзывов (glass, структура):**
```
Аватар:     [Фото человека — заглушка]
Имя:        [Полное имя]
Должность:  [Должность, Компания]
Цитата:     "[Конкретная проблема] + [что сделано] + [измеримый результат]"
```

**Важно при заполнении:** Никогда не использовать общие фразы типа "Отличная работа, рекомендую". Только конкретика с цифрами.

**Задача:** Убрать последнее трение и вызвать действие.

**Контент:**

- **Сигнал доступности:** `"Сейчас беру новых клиентов"`
- **Три кнопки:**
  1. **Telegram** (primary, neon violet, magnetic) — `"Написать в Telegram"`
  2. **WhatsApp** (secondary, outline) — `"Написать в WhatsApp"`

**Текст кнопок:** Никогда не "Отправить" или "Связаться". Конкретный глагол действия.

### Блок 7 — FAQ (Accordion)

**5 вопросов:**

1. **Сколько занимает проект?**
   > Зависит от задачи. Простая автоматизация — 3–5 дней. Лендинг — 2–3 недели. Сложная AI-система — 4–8 недель. На первом звонке скажу точнее.

2. **Сколько стоит?**
   > Проекты стартуют от небольших задач до комплексных систем. Обсуждаем бюджет на первом звонке — лучше скажу "это не мой формат", чем возьму проект и не выдам результат.

3. **Я не знаю точно, что мне нужно. С чего начать?**
   > С бесплатного звонка. Расскажи, что тебя тормозит в работе — вместе разберёмся, что решит проблему: автоматизация, сайт или AI-агент.

4. **Работаешь с клиентами из других городов и регионов?**
   > Да, работаю полностью удалённо. Клиенты из разных городов России.

5. **Что происходит после первого звонка?**
   > Я пришлю краткое резюме разговора и предложение: что делаем, сколько стоит, когда начинаем. Никакого давления — ты решаешь в своём темпе.

## 5. Анимации и взаимодействия

### Загрузка страницы
| Элемент | Анимация | Задержка |
|---------|----------|---------|
| H1 | font-weight 100→900 | 0s |
| Подзаголовок | fade-in translateY | 0.5s |
| typed.js старт | — | 0.8s |
| CTA кнопка | fade-in translateY | 1.0s |
| Частицы фона | fade-in | 0.3s |

### Scroll-driven (нативный CSS, без JS)
```css
@keyframes fadeUp {
  from { opacity: 0; transform: translateY(40px); }
  to   { opacity: 1; transform: translateY(0); }
}
.section-block {
  animation: fadeUp linear both;
  animation-timeline: view();
  animation-range: entry 0% cover 30%;
}
```
- Прогресс-бар вверху страницы
- Последовательное появление карточек (stagger 100ms)
- Clip-path wipe для изображений кейсов

### Hover-эффекты
| Элемент | Эффект |
|---------|--------|
| CTA кнопки | Magnetic (притяжение 30%) |
| Карточки услуг/кейсов | VanillaTilt 3D + фиолетовое свечение снизу |
| Bento карточки | translateY(-4px) + rotate(0.5deg) |
| Навигационные ссылки | Underline slide left→right |
| FAQ аккордион | Плавное раскрытие max-height |

### Кастомный курсор
```css
* { cursor: none; }
.cursor-dot {
  width: 12px;
  height: 12px;
  background: var(--accent-1);
  border-radius: 50%;
  position: fixed;
  pointer-events: none;
  mix-blend-mode: difference;
  transition: transform 0.1s;
}
```
- На hover CTA: увеличивается до 40px, появляется текст "→"
- На мобильных: отключён полностью (`pointer: fine` media query)

### Spotlight за курсором
```css
.hero::before {
  background: radial-gradient(
    600px circle at var(--mouse-x) var(--mouse-y),
    rgba(124, 58, 237, 0.15),
    transparent 70%
  );
}
```

### Доступность
- `prefers-reduced-motion`: все анимации отключаются, остаётся только opacity
- Все интерактивные элементы достижимы с клавиатуры
- Контраст текст/фон: минимум 4.5:1 (WCAG AA)

## 6. Мобильная адаптация

| Breakpoint | Изменения |
|-----------|-----------|
| `< 768px` | Bento Grid → 1 колонка; карточки услуг → вертикальный стек; CTA кнопка → 100% ширина |
| `< 480px` | H1 размер уменьшается; отступы сужаются; typed.js скорость замедляется |

**Мобильные правила:**
- CTA кнопка всегда доступна большим пальцем (min-height: 48px)
- Кастомный курсор отключён на touch-устройствах
- Все анимации облегчены (меньше трансформаций)
- Тест на 4G throttling: LCP < 2.5s
